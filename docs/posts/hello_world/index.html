Here, I showcase several ways to write a “Hello world” program in some of my favorite programming languages.

Scheme

(display "Hello world!")

Python

print("Hello world!")

Rust

fn main() {
   print!("Hello world!");
}

C

#include<stdio.h>
int main(){
  printf("\nHello world!");
  return 0;
}

C++

#include <iostream>
int main() {
  std::cout << "Hello world!\n";
  return 0;
}
