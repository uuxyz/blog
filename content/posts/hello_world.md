Here, I showcase several ways to write a "Hello world" program in some of my favorite programming languages.
## Scheme
```scheme
(display "Hello world!")
```

## Python
```python
print("Hello world!")
```

## Rust
```rust
fn main() {
   print!("Hello world!");
}
```

## C
```c
#include<stdio.h>
int main(){
  printf("\nHello world!");
  return 0;
}
```

## C++
```cpp
import module std;
int main() {
  std::print("Hello world!\n");
}
```
