Here, I showcase several ways to write a "Hello world" program in some of my favorite programming languages.
## scheme
```scheme
(display "Hello world!")
```

## python
```python
print("Hello world!")
```

## rust
```rust
fn main() {
   print!("Hello world!");
}
```

## c
```c
#include<stdio.h>
int main(){
  printf("\nHello world!");
  return 0;
}
```

## c++
```cpp
import module std;
int main() {
  std::print("Hello world!\n");
}
```
## java
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
